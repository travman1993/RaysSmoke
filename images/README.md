# Image Assets Needed

Drop your real photos into this folder using the exact filenames below — the site is already wired up to them. Recommended format: `.jpg`, optimized/compressed for web (aim under ~300KB each).

## Hero
- `trailer-hero.jpg` — wide, cinematic shot of the smoker trailer (ideally at dusk/with smoke). ~1920×1080 or larger.

## Favicon / Logo
A custom SVG flame-and-smoke mark is already built inline in `index.html` and at `icons/favicon.svg`. Swap it for a real logo file if you have one — just replace the `<svg class="brand__mark">` blocks with an `<img>` tag.
